# `foxclaw setup`

Initialize `~/.foxclaw/foxclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
foxclaw setup
foxclaw setup --workspace ~/.foxclaw/workspace
```

To run the wizard via setup:

```bash
foxclaw setup --wizard
```
